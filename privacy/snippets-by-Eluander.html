Política de Privacidade

Esta extensão armazena snippets de texto localmente no navegador do usuário utilizando a API chrome.storage.

Nenhum dado é enviado para servidores externos, compartilhado com terceiros ou utilizado para fins de rastreamento.

Todos os dados permanecem armazenados apenas no dispositivo do usuário.

A extensão não coleta:

  • dados pessoais
  • histórico de navegação
  • localização
  • credenciais
  • informações financeiras
